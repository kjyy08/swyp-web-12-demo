# Designer Implementation Plan

## 담당 에이전트: ui-designer
- **Type**: `oh-my-claudecode:designer`
- **작업 디렉토리**: `docs/design/`

## 역할
- 방명록 앱의 UI/UX 설계 및 디자인 명세 문서 작성
- frontend-dev가 구현 시 참조할 디자인 명세 제공

## 산출물
- `docs/design/design-guide.md` - 디자인 명세 문서
  - 색상 팔레트 (Primary, Secondary, Background, Text 등)
  - 타이포그래피 (폰트, 크기, 굵기)
  - 컴포넌트별 레이아웃 명세 (Header, Form, Card, List)
  - 반응형 브레이크포인트

## 작업 흐름
1. backend-plan.md의 Data Model 확인 (닉네임, 직군, 메시지 등)
2. 전체 페이지 레이아웃 설계
3. 컴포넌트별 디자인 명세 작성
4. 팀 리더에게 완료 보고

## 디자인 방향
- 모던하고 깔끔한 디자인
- 행사 분위기에 맞는 밝고 친근한 톤
- 카드형 레이아웃
- 반응형 (모바일 우선)
